Tips

  • Tags: Firefox's internal tag database isn't available to extensions, so this add-on treats #word tokens inside a bookmark's title as tags. Edit a bookmark's name to add or remove them.
  • Folders: A plain name (Work) matches any folder with that name in the bookmark's ancestry. Use a slash (Toolbar/Work/Docs) to match a specific path fragment.
  • Keywords: Simple case-insensitive substring match against the bookmark's title and URL.
